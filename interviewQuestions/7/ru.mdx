    title: 'Как удалить дубликаты в результате SQL-запроса?'
    description: 'Удаление дубликатов в SQL выполняется с помощью ключевого слова DISTINCT. Узнайте, как использовать его для получения уникальных записей.'
---

Использовать ключевое слово `DISTINCT` в операторе `SELECT`, чтобы вернуть только уникальные записи.

Пример:

```sql
SELECT DISTINCT position
FROM employees;
```
