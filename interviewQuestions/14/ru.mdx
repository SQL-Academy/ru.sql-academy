Оператор `LIKE` используется в условиях `WHERE` для поиска строк, соответствующих определенному шаблону.
В шаблонах используются подстановочные символы:

-   `%` — соответствует любой последовательности символов (включая нулевую длину).
-   `_` — соответствует одному любому символу.

## Примеры использования:

1. Поиск строк, начинающихся с «А»:

```sql
SELECT * FROM employees WHERE name LIKE 'А%';
```

2. Поиск строк, оканчивающихся на «e»:

```sql
SELECT * FROM employees WHERE name LIKE '%e';
```

3. Поиск строк, где второй символ — «a»:

```sql
SELECT * FROM employees WHERE name LIKE '_а%';
```
